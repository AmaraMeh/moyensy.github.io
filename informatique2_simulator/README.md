# Simulateur Informatique 2

Ce dossier contient le simulateur d'examens interactif pour la matière Informatique 2 (Pascal, tableaux, sous-programmes, etc.).

## Structure
- `index.html` : Page d'accueil du simulateur, login, choix des examens.
- `exams/` : Dossier pour les 5 examens (questions, déroulement, etc.).
- `amaramehdinaya2025.html` : Panneau d'administration (accès par lien direct).
- `js/` : Scripts JS (logique principale, admin, etc.).
- `css/` : Styles additionnels si besoin.
- `firebase.js` : Configuration et initialisation Firebase.
- `assets/` : Images, icônes, etc.

## Fonctionnalités
- Login par pseudo + PIN 4 chiffres
- 5 examens interactifs (accès selon planning)
- Classement et stats live pour le premier examen
- Mode sombre/clair, animations, responsive
- Panneau admin pour gestion des examens et stats
